# Tier e Funzionalità

Questa pagina descrive i 3 livelli di personalizzazione del template Palestra, con costi, effort, e features specifiche.

## Tier Base — €500–800 (70–80 ore)

**Ideal per**: PMI che cercano sito vetrina professionale senza booking online.

### Descrizione
Sito statico SSR pre-renderizzato con home, about, catalogo lezioni, orari, FAQ e contatti. Zero complessità backend.

### Per chi
- Palestre in fase startup senza budget IT
- Rebranding di sito legacy verso modern stack
- Showcase tecnico per demo commerciale

### Funzionalità incluse
- Hero CTA "Prenota una lezione"
- About + team con foto istruttori
- Catalogo lezioni (cardio, pesi, yoga) con prezzi €15–25
- Orari settimanali tabella responsive
- Galleria 20–30 foto attrezzature/ambienti
- Testimonial clienti
- Form contatto con email Resend
- Blog wellness articoli (2–3 di esempio)
- Schema.org JSON-LD (`HealthClub`)
- Lighthouse Score ≥90 SEO, ≥85 Performance

### NON incluse
- Prenotazioni online calendario
- Area clienti riservata
- Integrazione wearable
- App mobile
- AI features

### Add-on consigliati
- Aggiunta 20 articoli blog wellness (+€150)
- Multi-lingua IT/EN (+€200)
- Google My Business integration (+€100)

### Manutenzione mensile
- Aggiornamento orari lezioni: 30 min
- Aggiornamento prezzi: 15 min
- Monitoring Lighthouse score: mensile autonomo
- **Costo**: €0 self-service oppure €50/mese con supporto Federico

## Tier Intermedio — €1.500–2.200 (230–250 ore)

**Ideal per**: Palestre consolidate che vogliono booking online + CRM.

### Descrizione
Tier Base + prenotazioni online FullCalendar, login utente, area personale, gestione abbonamenti admin, notifiche SMS/email, Stripe Checkout per pagamenti.

### Per chi
- Palestre con 200+ clienti attivi
- Studio pilates/yoga con corsi regolari
- Operatori che volevano SaaS verticale ma cercano indipendenza

### Funzionalità incluse
- Tutto il Tier Base
- **Prenotazioni online**: FullCalendar con slot 30–60 min, sync backend in real-time
- **Area cliente**: login via email, storico lezioni, profilo, preferenze istruttore
- **Gestione abbonamenti admin**: pacchetti (mensile illimitato €49, 10 lezioni €79, pay-per-class €15)
- **SMS/Email reminder**: automatici 24h + 2h prima lezione
- **Pagamenti Stripe**: checkout integrato, rateale opzionale
- **CRM clienti base**: lista prenotazioni, statistiche presenze, no-show tracking
- **Newsletter Resend**: invii promo a iscritti
- **Multi-lingua IT/EN**
- **GDPR**: consensi digitalizzati, cookie banner, privacy policy template
- Schema.org esteso (Event, Offer)

### NON incluse
- Schede allenamento personalizzate
- Wearable integration
- AI raccomandazioni
- App mobile
- Analytics avanzate

### Add-on consigliati
- Estensione CRM con segmentazione cliente (+€300)
- Integrazione Google Calendar istruttori (+€150)
- SMS broadcast promo illimitati (+€100/mese)
- Whatsapp reminder via Twilio (+€200)

### Manutenzione mensile
- Gestione prenotazioni no-show: 1h
- Aggiornamento listino: 30 min
- Supporto clienti tech: 2h stimate
- **Costo**: €100/mese o modello one-shot pre-anno €1.200

## Tier Avanzato — €4.000–6.000 (480–500 ore)

**Ideal per**: Palestre/chain multi-sede che vogliono AI + mobile app.

### Descrizione
Tier Intermedio + schede allenamento personalizzate PDF, AI workout recommender con Ollama (llama3.1:8b), integrazione wearable (Apple Health, Garmin, Fitbit), app mobile React Native, analytics dashboard, gamification badge/leaderboard.

### Per chi
- Chain palestre multi-sede (3+ sedi)
- Palestre wellness premium con personal trainer
- Startup tech che usa sito come MVP per app futura

### Funzionalità incluse
- Tutto il Tier Intermedio
- **Schede allenamento personalizzate**: PDF generati per cliente basato su goal (dimagrimento, ipertrofia, endurance)
- **AI workout recommender**: Ollama llama3.1:8b suggerisce esercizi personalizzati
- **Wearable sync**: Apple Health, Garmin Connect, Fitbit API per HR/steps
- **App mobile React Native**: iOS/Android con offline support, timer esercizi, progress tracking
- **Analytics dashboard admin**: retention rate, revenue per corso, client lifetime value
- **Gamification**: badge (primo mese, 10 lezioni, PR strength), leaderboard settimanale, achievement unlock
- **SMS broadcast avanzate**: promo geolocalizzate (clienti sedi specifiche)
- **Multi-sede**: gestione abbonamenti per sede, riepilogo fatturato consolidato

### NON incluse
- Integrazione Stripe Connect (multi-split per franchisee)
- Video esercizi custom

### Add-on consigliati
- Video esercizi da biblioteca cloud (+€800)
- Integrazione Strava/Fitbit avanzata (+€400)
- Dashboard white-label per franchisee (+€600)
- Supporto tecnico 24/5: €500/mese

### Manutenzione mensile
- Monitoring Ollama health: 30 min
- Aggiornamento dataset AI: 2h
- Supporto clienti avanzato: 4h stimate
- **Costo**: €200/mese (incluso 10h supporto)

## Processo scelta tier

1. **Inventario attuale**: Quanti clienti? Che prenotazioni? App? AI?
2. **Roadmap**: Dove vuoi essere in 12 mesi?
3. **Budget IT**: One-time vs recurring subscription preference?
4. **Supporto**: Preferisci self-service o managed?
